prcptot_EnsembleStats files are based on annual precipitation totals and give either the 30-year (1985-2014) average  (Avg_ann) or the 30-year maximum (Max_ann) values based on the 16-model downscaled LOCA2 or STAR ensemble means.

rx1day_EnsembleStats files are based on the maximum daily precitation total in each year and give either the 30-year (1985-2014) average (Avg_ann) or the 30-year maximum (Max_ann) values based on the 16-model downscaled LOCA2 or STAR ensemble mean.

prcptot_CalcTimeStats1 files are analogous to the prcptot_EnsembleStats files but use data from Livneh or nClimGrid.  These files incoporate both the Avg_ann and Max_ann values.

rx1day_CalcTimeStats1 files are analogous to the rx1day_EnsembleStats files but use data from Livneh or nClimGrid.  These files incoporate both the Avg_ann and Max_ann values.

Get_ACIS.py allows access to the observed GHCN station data.
